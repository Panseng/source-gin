# About
这是一个`gin`的实践+源码讲解项目

## 使用 practise-code
```bash
# 获取项目代码
git clone https://github.com/Panseng/source-gin

# 进入`practise-code`文件夹
cd source-gin/practise-code

# 依赖包安装
go mod tidy

# 运行项目源代码
go run main.go

# 编译
go build -o server.exe main.go

# 运行二进制
server.exe
```

`practise-code`详细内容见：[readme](practise-code/readme.md)
